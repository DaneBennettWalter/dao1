# DAO1

Own together. Decide together. Build together.

Tools for shared ownership and collective decision-making.

## About

DAO1 is a set of tools that let people own things together and make decisions together — without needing lawyers, landlords, or a computer science degree to do it.

## Live Projects

- **Hotel Brendle** — 67-room historic hotel, Robstown TX
- **Homestead San Marcos** — 73-acre sustainable community, Kingsbury TX
- **Robstown House of Prayer** — Community food bank and worship center
- **SpaceCamp TX** — Space education and community events
- **Deep Tech Stir Fry** — Roving event series for brilliant minds
- **CBBTX** — Construction and labor engine
